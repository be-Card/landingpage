# beCard Landing (Web)

## Desarrollo

```bash
cd landing-web
npm install
npm run dev
```

## Notas

- La landing usa los assets existentes en `.figma/image` sirviéndolos como estáticos bajo `/image/*` mediante la configuración de Vite.
- El layout está pensado para 1440px (Figma) y se escala automáticamente en pantallas más chicas.
